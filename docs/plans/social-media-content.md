## Instagram Strategy Overview

### Content Pillars
1. **App Education** - How it works, features, the reveal
2. **Relatable ADHD/Procrastination** - Memes, "me too" moments
3. **Tick & Personality** - Brand character, notifications, voice

### Posting Schedule
- **Grid Posts:** 3 per week (Mon, Wed, Fri recommended)
- **Reels:** 2-3 per week (Tues, Thurs, Sat)

### Grid Aesthetic
Alternating pattern:
1. App screenshot / UI showcase
2. Text graphic (notification quote, brand colors)
3. Carousel (educational or entertaining)
4. Repeat

Use the brand palette (Lavender Dream, Soft Peach, Hot Pink, Mint) for text graphics. Keep it cohesive but not boring.
